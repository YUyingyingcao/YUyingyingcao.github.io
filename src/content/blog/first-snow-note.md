这个站点想留一点慢慢翻页的感觉。

它不是用来严肃宣布什么的地方，更像把一天里捡到的小东西放进抽屉：一张图、一段代码、一点想法、一个还没打磨好的小玩具。

## 小站的气味

- 首页像雪地，先让人停下来。
- 手札页放文章，技术和生活都可以写。
- 工房页放实验品，不要求每个都完整。
- 关于页不用把人介绍得太用力，像贴在门后的便签就好。

## 写东西的节奏

以后文章可以长一点，也可以很短。重要的是每次打开时，它看起来仍然像自己的房间。

```ts title="tiny-note.ts"
const note = {
  weather: "snow",
  pace: "slow",
  rule: "把喜欢的东西认真放好",
};

console.log(`${note.weather} / ${note.pace}`);
```

如果某天写教程，代码块、表格、标题锚点都已经准备好了。

| 区域 | 放什么 |
| --- | --- |
| Blog | 长文、教程、日记 |
| Projects | 小工具、作品、实验 |
| About | 个人碎片和联系方式 |

> 雪会慢慢盖住脚印，但不会取消走过的路。
